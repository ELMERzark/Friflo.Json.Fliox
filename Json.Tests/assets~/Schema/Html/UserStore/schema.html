Friflo.Json.Fliox.Hub.Host.Auth.Rights namespace

TaskRight abstract class

Each Role has a set of taskRights.
Each TaskRight is a rule used to grant or deny a specific database operation or command execution.
The database operation or command execution is granted if any of it taskRights grant access.
type discriminator right type
"dbFull" DbFullRight
"dbTask" DbTaskRight
"dbContainer" DbContainerRight
"sendMessage" SendMessageRight
"subscribeMessage" SubscribeMessageRight
"predicate" PredicateRight
description string | null optional description explaining the Right

DbFullRight class extends TaskRight

Allow full access to the given database.
type "dbFull"right type
database string a specific database: 'test_db', multiple databases by prefix: 'test_*', all databases: '*'

DbTaskRight class extends TaskRight

DbTaskRight grant database access by a set of task types.
type "dbTask"right type
database string a specific database: 'test_db', multiple databases by prefix: 'test_*', all databases: '*'
types TaskType[] set fo task types like: create, read, upsert, delete, query, ...

DbContainerRight class extends TaskRight

DbContainerRight grant database access for the given containers based on a set of operations.
E.g. create, read, upsert, delete, query or aggregate (count)
It also allows subscribing database changes by subscribeChanges
type "dbContainer"right type
database string a specific database: 'test_db', multiple databases by prefix: 'test_*', all databases: '*'
containers ContainerAccess[] grant execution of operations and subscriptions on listed containers

SendMessageRight allows sending messages to a database by a set of names.
