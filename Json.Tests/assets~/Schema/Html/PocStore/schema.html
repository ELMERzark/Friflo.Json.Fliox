DbSchema class

A DbSchema can be assigned to a database to specify its containers, commands and messages.
The types used by the Schema are declared within jsonSchemas.
The type referenced by the tuple schemaName / schemaPath specify the database containers, commands and messages.
id string database name
schemaName string refer a type definition of the JSON Schema referenced with schemaPath
schemaPath string refer a JSON Schema in jsonSchemas
jsonSchemas key âžž any map of JSON Schemas each containing a set of type definitions.
Each JSON Schema is identified by its unique path
